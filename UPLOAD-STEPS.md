GitHub Pages upload steps for the CSC test repository

1. Open the repository that is published at https://devnain21.github.io/CSC/
2. Delete the current source-code files from that test repository, or replace them completely.
3. Upload everything inside this github-pages-ready folder to the repository root.
4. Make sure GitHub Pages is enabled for that repository.
5. In repository Settings > Pages, use Deploy from a branch and select the branch/root where these files are uploaded.
6. Wait 1 to 3 minutes, then open https://devnain21.github.io/CSC/

Important

- Upload the contents of this folder, not the folder itself.
- This folder already contains index.html, _next, sitemap.xml, robots.txt, and .nojekyll.
- The site is built for the /CSC path.
- If the old README is still showing, GitHub Pages is still serving the old repository contents.
